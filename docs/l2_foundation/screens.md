# 画面設計（情報設計レベル）

> L2（`foundation.md`）から分離した**画面（ページ）構成**。各ページに **①コンポーネント配置＋大きさ（ASCIIワイヤー）** と
> **②動線** の 2 層を持たせる。細かい CSS・コンポーネント実装は L3 以降。
> **レスポンシブ必須**: 各画面にモバイル/デスクトップ両方のレイアウト感を注記する。

## 画面一覧（v1＝2画面）

**探すはトップ内で完結（バウンス無し）／別ページに飛ぶのは作品詳細だけ／詳細から同タグへ回遊** が肝。

| 画面 | 役割 | 対応 REQ |
|------|------|---------|
| **トップ（発見ハブ）** | 検索・今期・人気・新着・ぜんぶ見る（絞り込み/並び替え）を 1 ページで完結 | REQ-0001/0002/0003/0004/0008/0010 |
| **作品(シリーズ)詳細** | 各話一覧＋公式 deep-link＋同タグ/ジャンルへ回遊 | REQ-0001/0005（＋回遊で 0003/0008） |

いずれも実行時は**事前生成した用途別 JSON を読むだけ**（API 直叩きなし）。全状態は **URL クエリで再現**（共有・ブックマーク・戻り位置が予測どおり）。

## 全体動線

```
   入口
    │
    ▼
 ┌───────── トップ（発見ハブ） ─────────┐
 │ ヒーロー検索 / ヘッダ縮小検索 ──┐      │
 │   今期・人気・新着シリーズ・最新話 │      │   検索・絞り込み・並び替え
 │   （カルーセル）                │──────┼─▶ 「ぜんぶ見る」を同ページ更新（遷移しない）
 │   ぜんぶ見る（グリッド＋ページング）│      │   状態は URL クエリ
 └───────┬───────────────────────┘      │
         │ カード click（シリーズ）              │
         ▼                                     │
 ┌───── 作品(シリーズ)詳細 ─────┐                │
 │ 各話 [▶公式] ─────────────────┼─▶ 外部離脱（nicovideo.jp/watch/<contentId>）
 │ 同じタグ/ジャンルへ ───────────┼─▶ トップの「ぜんぶ見る」を該当フィルタ状態で開く（回遊）
 │ 戻る ────────────────────────┼─▶ トップ
 └──────────────────────────────┘
```

セクションはすべて**横スクロールのカルーセル**、最後の「ぜんぶ見る」だけ縦グリッド。カードは原則**シリーズ単位**（最新話のみ話単位）。

### レスポンシブ注記

- **ぜんぶ見る グリッド列数は画面幅で可変**: モバイル **2列** 〜 タブレット 3列 〜 デスクトップ **4〜6列**。
- **カルーセル**は全幅でも横スクロール。モバイルは**横スワイプ**＋スナップ。はみ出しは `‹ › / → →` で示唆。
- **ヒーロー検索はモバイルでも上部**に出す。スクロールで**ヘッダ縮小検索（sticky）**へ畳むのは両幅共通。
- ブレークポイント（目安・確定は L3）: `<640px`=モバイル(2列)、`640–1024px`=3列、`>1024px`=4〜6列。
- フィルタ行はデスクトップ横一列、モバイルは折返し or ドロワー。

### ② 動線

- **入口 → トップ**。まず目に入るのはヒーロー検索（検索ファースト）。
- **カルーセル（今期/人気/新着シリーズ/最新話）でカード click → 作品詳細**（最新話カードは該当シリーズの詳細へ、その話に着地）。
- **ヒーロー検索／ヘッダ縮小検索に入力 →「ぜんぶ見る」が結果更新（同ページ・遷移しない）**。該当セクションへスクロール。
- **絞り込み[ジャンル/タグ/クール]・並び替え（再生数/新着）→「ぜんぶ見る」更新（同ページ）**。件数・適用中チップを更新。
- すべて **URL クエリに反映**（例 `?q=…&genre=…&tag=日常&sort=new&page=2`）。戻る/共有/ブックマークで同じ状態を再現。
- **無限スクロールにしない**＝ページングで戻り位置が予測どおり。

## 2. 作品（シリーズ）詳細

**別ページに飛ぶのはここだけ。** 各話一覧と公式への導線＋同タグ/ジャンルへの回遊。視聴本体は扱わない。

### ① ワイヤー（デスクトップ）

```
┌──────────────────────────────────────────────────────────────┐
│ [ロゴ]   [🔍 検索…]                          [← 戻る]          │ ← ヘッダ(sticky検索)
├──────────────────────────────────────────────────────────────┤
│ ┌──────────┐   作品タイトル                                   │
│ │          │   ジャンル: アニメ                                │  シリーズ・メタ
│ │  THUMB   │   タグ: [日常] [ほのぼの] [2026春] …             │  ← タグはチップ
│ │ (large)  │   [▶ 公式シリーズページ →]                       │  REQ-0005(/series/<id>)
│ └──────────┘                                                  │
├──────────────────────────────────────────────────────────────┤
│ 各話 (12話)                                  並び: (●話数)(○再生)│  REQ-0001
│ ──────────────────────────────────────────────────────────── │
│ #1   タイトル…                    ▶ 12,345    [▶ 公式]         │  各話→watch/<contentId>
│ #2   タイトル…                    ▶  9,876    [▶ 公式]         │  REQ-0005
│ #3   タイトル…                    ▶  8,765    [▶ 公式]         │
│ …                                                            │
├──────────────────────────────────────────────────────────────┤
│ ▸ 同じタグ/ジャンルへ                                          │  回遊（REQ-0003/0008）
│ [日常 →] [ほのぼの →] [2026春 →] [ジャンル:アニメ →]            │  → トップの「ぜんぶ見る」を
│                                                              │    該当フィルタ状態で開く
└──────────────────────────────────────────────────────────────┘
```

### ① ワイヤー（モバイル・簡易）

```
┌──────────────────┐
│[ロゴ][🔍] [←戻る] │
├──────────────────┤
│ ┌──────────────┐ │  サムネは上に大きく、
│ │    THUMB     │ │  メタは下に縦積み
│ └──────────────┘ │
│ 作品タイトル      │
│ ジャンル: アニメ  │
│ [日常][ほのぼの]  │  ← タグチップ折返し
│ [▶ 公式シリーズ →]│
├──────────────────┤
│ 各話 (12話) (並▾) │
│ #1 …    ▶12,345  │
│        [▶ 公式]  │  ← 1話=2段でも可
│ #2 …    ▶ 9,876  │
│        [▶ 公式]  │
│ …                │
├──────────────────┤
│ ▸ 同じタグ/genreへ│
│ [日常→][2026春→]  │  ← 横スワイプ
└──────────────────┘
```

### レスポンシブ注記

- デスクトップは**サムネ左・メタ右の横並び**、モバイルは**サムネ上・メタ下の縦積み**に切替。
- 各話リストは全幅。モバイルは「話名」と「再生数＋[▶公式]」を 2 段に折返してよい。
- タグチップ・回遊リンクは折返し（デスクトップ）／横スワイプ（モバイル）。

### ② 動線

- **トップのカード click → ここに着地**（最新話カード経由なら当該話にスクロール）。
- **各話 [▶ 公式] → 外部離脱**（`nicovideo.jp/watch/<contentId>`／会員視聴）。シリーズ見出し [▶ 公式シリーズページ] → `/series/<id>`。
- **[同じタグ/ジャンルへ] → トップの「ぜんぶ見る」を該当フィルタ状態（URL クエリ）で開く**＝回遊。
- **[← 戻る] → トップ**（直前の状態を URL で復元）。

## 新着の2種（データ注記）

REQ-0010 は 2 系統に割れる。ビルド時に **snapshot（話＝`contentId` 単位）を list.json の series id で束ねて**両方を静的 JSON 化する。

| 種別 | 定義 | カード単位 | 配置 |
|------|------|-----------|------|
| **新着シリーズ** | シリーズの初出日（束ねた各話 `startTime` の最小）が新しい | シリーズ | §3 |
| **最新の動画** | 話の `startTime` 最大（既存作の新着話を含む） | 話 | §4 |

## REQ 対応（再掲）

- トップ: 検索/ぜんぶ見る=REQ-0001、人気=REQ-0002、ジャンル=REQ-0003、クール=REQ-0004、タグ=REQ-0008、
  新着（**新着シリーズ＋最新話の 2 系統**）=REQ-0010。
- 作品詳細: 各話一覧=REQ-0001、deep-link=REQ-0005、同タグ/ジャンル回遊=REQ-0003/0008。
