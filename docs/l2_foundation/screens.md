# 画面設計（情報設計レベル）

> L2（`foundation.md`）から分離した**画面（ページ）構成**。レイアウトレベルの「どこに何を出すか」までを示す。
> 細かい CSS・コンポーネント設計・実装は L3 以降。

## 画面一覧（v1＝3画面）

ランキング/新着/ジャンル/タグ/クールは**独立ページにしない**。これらは「一覧」画面の**並び替え・絞り込みの状態**にすぎない。
v1 は次の 3 画面で足りる。

| 画面 | 役割 | 対応 REQ |
|------|------|---------|
| **トップ** | 入口。人気と新着を軽く見せ、一覧へ送る | REQ-0002 / REQ-0010 |
| **一覧** | 検索・絞り込み（ジャンル/タグ/クール）・並び替え（再生数順/新着順）を内包 | REQ-0001/0002/0003/0004/0008/0010 |
| **作品(シリーズ)詳細** | 各話一覧と公式プレイヤーへの deep-link | REQ-0001 / REQ-0005 |

いずれも実行時は**事前生成した用途別 JSON を読むだけ**（API 直叩きなし）。

---

## 1. トップ

人気・新着を「軽く」見せ、深掘りは一覧へ。

```
┌──────────────────────────────────────────────┐
│ [ロゴ] nico-danime-viewer        [検索 🔍]      │  ← ヘッダ（全画面共通）
├──────────────────────────────────────────────┤
│  ▸ 再生数ランキング              [一覧で見る →] │  REQ-0002
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐        │
│  │card│ │card│ │card│ │card│ │card│  …上位   │  サムネ+タイトル+再生数
│  └────┘ └────┘ └────┘ └────┘ └────┘        │
├──────────────────────────────────────────────┤
│  ▸ 新着（最新の動画）            [一覧で見る →] │  REQ-0010
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐        │
│  │card│ │card│ │card│ │card│ │card│  …新着   │  サムネ+タイトル+投稿日
│  └────┘ └────┘ └────┘ └────┘ └────┘        │
├──────────────────────────────────────────────┤
│  ▸ 探す: [ジャンル] [タグ] [クール]            │  一覧の絞り込み状態への入口
└──────────────────────────────────────────────┘
```

- 「一覧で見る →」「ジャンル/タグ/クール」は、対応する**絞り込み/並び替え状態の一覧**へ遷移するだけ。
- 使うデータ: ランキングJSON（`viewCounter` 上位）、新着JSON（`startTime` 降順）。

## 2. 一覧

検索・絞り込み・並び替えを 1 画面に内包。ランキング/新着/ジャンル別/タグ別/クール別は**ここの状態**。

```
┌──────────────────────────────────────────────┐
│ [ロゴ]                            [検索 🔍 ___] │  ← ヘッダ
├──────────────────────────────────────────────┤
│ 絞り込み: [ジャンル▾] [タグ▾] [クール▾]         │  フィルタ（REQ-0003/0008/0004）
│ 並び替え: (●再生数順) (○新着順)                 │  ソート（REQ-0002/0010）
│ 件数: 1,234件 / 適用中: タグ「日常」 [×]        │  現在の状態を明示（予測どおり）
├──────────────────────────────────────────────┤
│ ┌────┐ ┌────┐ ┌────┐ ┌────┐                │
│ │card│ │card│ │card│ │card│   作品カードの   │  サムネ+タイトル+再生数/投稿日+タグ
│ └────┘ └────┘ └────┘ └────┘   グリッド       │  カード → 作品詳細へ
│ ┌────┐ ┌────┐ ┌────┐ ┌────┐                │
│ │card│ │card│ │card│ │card│                 │
│ └────┘ └────┘ └────┘ └────┘                │
├──────────────────────────────────────────────┤
│              [← 前]  2 / 57  [次 →]            │  ページング
└──────────────────────────────────────────────┘
```

- フィルタ・ソートは URL クエリに反映（同じ状態は同じ URL ＝予測どおり・共有可能）。
- 使うデータ: 全作品インデックスJSON＋用途別索引（ジャンル/タグ/クール）。検索はクライアント側。

## 3. 作品（シリーズ）詳細

各話一覧と公式プレイヤーへの導線。視聴本体は扱わない。

```
┌──────────────────────────────────────────────┐
│ [ロゴ]                            [検索 🔍]     │  ← ヘッダ
├──────────────────────────────────────────────┤
│ ┌──────┐  作品タイトル                          │
│ │ thumb │  ジャンル / タグ: 日常 ほのぼの …      │  シリーズ・メタ
│ │      │  [▶ ニコニコ支店の公式ページで見る]    │  REQ-0005（シリーズへ deep-link）
│ └──────┘                                       │
├──────────────────────────────────────────────┤
│  各話                                          │  REQ-0001（list.json の series で束ねた各話）
│  ─────────────────────────────────────         │
│  #1  タイトル…           再生 12,345  [▶ 公式]  │  各話 → watch/<contentId> へ deep-link
│  #2  タイトル…           再生  9,876  [▶ 公式]  │  REQ-0005
│  #3  タイトル…           再生  8,765  [▶ 公式]  │
│  …                                            │
└──────────────────────────────────────────────┘
```

- [▶ 公式] は `nicovideo.jp/watch/<contentId>`、シリーズ見出しは `/series/<id>` へ。視聴は公式プレイヤー（会員）。
- 使うデータ: シリーズJSON（list.json の series id で束ねた各話＋ `viewCounter` 等）。
