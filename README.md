# nico-danime-viewer

**dアニメストア ニコニコ支店**の配信作品を、見やすく探せる**非公式の発見ページ**です。

🔗 **サイト（公開）**: https://emanon-i.github.io/nico-danime-viewer/
※ 公開は準備中です（サイトの公開後、上記 URL でご利用いただけます）。

> ⚠️ 個人が趣味で作っている**非公式・非営利**のツールです。ドワンゴ／dアニメストアとは一切関係ありません。

## これは何

dアニメストア ニコニコ支店で配信中のアニメを、**ランキング・タグ・ジャンル・放送クール**から
ザッピング感覚で探せる軽量な Web ページです。公式サイトでは埋もれがちな作品に出会いやすくすることが目的です。

**このサイトは動画を配信しません。** 作品を選ぶと**ニコニコの公式プレイヤーへ移動**します。

## 使い方

1. サイト（GitHub Pages）を開く。
2. **ランキング／タグ／ジャンル／クール**から作品を探す。
3. 気になった作品を選ぶと、**ニコニコ公式の作品ページ（公式プレイヤー）へ移動**して視聴できます。

## スコープと注意

- **対象はニコニコ支店のみ。** dアニメストア本店（NTT DOCOMO・`animestore.docomo.ne.jp`）は
  ラインナップも視聴プレイヤーも異なる**別サービス**で、このサイトの対象外です。
- **非公式・個人制作・非営利。** 公式の機能・サポートではありません。
- **視聴にはニコニコ支店の会員と公式プレイヤーが必要です。** 本サイトは発見の入口を提供するだけです。
- 作品データは**ニコニコの公開 API から日次で取得**しています（鮮度は最大 1 日程度ずれます）。

## 技術（ざっくり）

- 静的サイト（**Vite + TypeScript**）を **GitHub Pages** で配信。
- **GitHub Actions** が日次でニコニコの公開 API から作品データを取得し、静的 JSON に変換 → サイトが読み込み。
- ブラウザからは API を直接叩かず、あらかじめ用意した JSON だけを読むシンプルな構成です。

## 開発者向け

仕様は `docs/`（Tri-SSD: L0 要求 / L1 要件 / L2 構成）に、データ取得の API リファレンスは
`.claude/skills/nico-snapshot-api/` にあります。

```bash
npm install
npm run fetch    # ニコニコ公開APIから data/*.json を生成（User-Agent 必須）
npm run dev      # ローカル開発サーバ
npm run build    # 静的サイトをビルド（dist/）
```

> 現状は**土台のみ**で、各スクリプトは雛形（未実装）です。
> 取得データ（`data/*.json`）は生成物のため git 管理外です。再配布はしません。
